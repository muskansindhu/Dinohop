# Dinohop

Dinohop is a browser-based endless runner game. Help the dinosaur avoid obstacles and survive as long as possible!

## Features

- Simple UI built with **JavaScript** and **HTML**
- Responsive, fast gameplay

## Getting Started

1. Clone the repository:
   ```
   git clone https://github.com/muskansindhu/Dinohop.git
   ```
2. Install dependencies:
   ```
   npm install
   ```
3. Start the game:
   ```
   npm run start
   ```
4. Open your browser and go to [http://localhost:8000](http://localhost:8000)

## How to Play

- Press the **spacebar** or **click/tap** to jump.
- Avoid obstacles!
- Try to beat your high score as the game speeds up.
